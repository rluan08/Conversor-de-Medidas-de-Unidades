# Conversor de Unidades

Aplicacao web para conversao de diferentes unidades de medida.

## Funcionalidades

- Conversao de comprimento
- Conversao de massa
- Conversao de temperatura
- Conversao de velocidade
- Conversao de area
- Conversao de armazenamento digital

## Tecnologias

- Python (backend)
- HTML
- CSS
- JavaScript

## Como executar

1. Instale as dependencias do Python
2. Execute o backend
3. Abra o arquivo index.html
